The Analyzers

Mythril

Adapted from the repository's readme:
Mythril OSS is the classic security analysis tool for Ethereum smart contracts. It uses concolic analysis, taint analysis and control flow checking to detect a variety of security vulnerabilities.

Manticore

Manticore is a symbolic execution tool for analysis of binaries and smart contracts.

Adapted from this: Manticore is not really intended for lightweight detection of certain kinds of vulnerabilities it was designed for arbitrary discovery of real possible contract states, using a transactional approach every state should be reproducible, no false positives in the sense that no infeasible states are generated this is more useful for contract verification part of the secure development process, rather than lightweight developer security feedback.

Oyente

Adapted the main Oyente homepage linked above:
We built a symbolic execution tool to find potential security bugs in contracts written by developers for the existing Ethereum system. Our objective for Oyente is to increase the security of the smart contracts by identifying vulnerabilities which will allow developers to mitigate the identified risks.

Oyente is integrated into remix for analysis.
